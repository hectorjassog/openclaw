# Summarize Skill

Fast CLI to summarize URLs, local files, and YouTube links.

## When to Use

- "What's this link about?"
- "Summarize this URL/article"
- "Transcribe this YouTube video"

## Commands

```bash
summarize "https://example.com"
summarize "/path/to/file.pdf"
```
